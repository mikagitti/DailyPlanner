This have

-    json-server where can be used to API call
-    UI made by NextJs

# How to use:

## 1. Open Terminal and Run the json server:

```bash
npm run json-server
```

## 2. Open new Terminal and Run the development server:

```bash
npm run dev
```

## 3. Open browser and type:

[http://localhost:3000](http://localhost:3000)
